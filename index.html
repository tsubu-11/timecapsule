プライバシーポリシー

最終更新: 2025年

1. はじめに

タイムカプセル（以下「本アプリ」）は、あなたのプライバシーを尊重します。本ポリシーは、本アプリが扱う情報について説明します。

2. 収集・保存する情報

本アプリが保存する情報は以下のみです。

  • カプセルのタイトル・本文・作成日・開封日
  • アプリの設定（テーマ・文字サイズ・通知設定など）

これらの情報はすべてお使いのデバイスのブラウザストレージ（localStorage）にのみ保存されます。

3. 外部への送信

本アプリはいかなる情報も外部のサーバーや第三者に送信・提供しません。インターネット接続なしでも動作します（初回ロード後）。

4. データの管理と削除

  • 設定画面の「バックアップを書き出す」でデータをJSONファイルとして保存できます
  • ブラウザのサイトデータ消去を行うと、すべてのカプセルが削除されます
  • 定期的なバックアップを強くお勧めします

5. 通知

通知を許可した場合、カプセルの開封日にデバイスの通知機能を使用します。通知の許可はいつでもデバイスのOS設定から変更できます。

6. お問い合わせ

ご不明な点は GitHub リポジトリ（github.com/tsubu-11/timecapsule）にてお問い合わせください。
